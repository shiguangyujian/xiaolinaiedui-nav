教学工具导航

— 管理后台：添加新资源 —

第 1 步：放入资源文件
如果你有一个做好的互动课件（HTML 文件）或其他资源，请将它复制到 edui-nav/resources/ 文件夹中。
文件命名建议：使用纯英文或拼音，不要有空格和特殊字符。
例如：fenshuqiang.html、yuan_derenshi.html

如果只是添加一个外部链接（比如别人的网站），跳过这步。
第 2 步：选择区块
第 3 步：填写工具信息
链接格式说明：
· 外部链接：https://www.xxx.com
· 本地资源：resources/你的文件名.html（需先完成第1步）
· 如果还没准备好链接，填 # 占位
第 4 步：生成并复制
已复制到剪贴板！
如何使用生成的代码：
1. 点击"生成 JSON 代码"按钮
2. 点击"一键复制"按钮
3. 打开 edui-nav/data/tools.json 文件
4. 找到对应区块的 "tools": [ 部分
5. 在 ] 前粘贴新代码（注意逗号分隔）
6. 保存文件，刷新首页即可看到新工具

如果你是用 VS Code 编辑：在 data/tools.json 中找到 "tools": [，在该数组的最后一个工具后面加逗号，然后粘贴即可。
